Answer: sessionStorage is similar to localStorage;

the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends.

in session storage data is deleted when the browser is closed and data is not available for future sessions.

in local storage, data is not deleted when the browser is closed and data is available for future sessions

Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.
